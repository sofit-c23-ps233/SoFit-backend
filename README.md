## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* node.js

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sofit-c23-ps233/SoFit-backend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
